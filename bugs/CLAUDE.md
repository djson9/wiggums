# Bugs

This directory contains bugs that we have, so that they may be investigated.

## Example
Example of what a ticket should look like.
<example>
---
Date: 2026-02-01
Title: ScriptPopup p.program is nil in Headless Mode
Status: completed
Description: User should be able to perform user desired action. We should verify that it works after implementing.
---
Did x, y, z actions and was 80% successful and 20% failed cases. << ALWAYS INCLUDE FAILED CASES
</example>


## Filing New Bugs
File new bugs to this directory, with the format.
[EPOCH SECONDS]\_[Title].md
1769879447_Title.md

Include reproduction steps, which gives the exact steps to reproduce the bug.
Also include a detailed description of the problem, and how it affects the user's objectives.
